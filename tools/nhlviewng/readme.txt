To view detailed information on installation and usage, launch nhlviewng.exe and consult the User's Guide.
You can also press F1 at any time to display contextual help on the selected element.

If you are having trouble viewing the help file, PDF User's Guide is also available:
http://www.artemkh.com/nhl/nhlviewng/

To have the preview of player appearance, you need to download graphical assets for your version of the game.
Go to http://www.artemkh.com/nhl/nhlviewng/, select the appropriate ZIP archive and extract it in the NHLView folder.
